# Free n8n Workflows

This folder contains free n8n workflow templates that you can import and use in your own n8n instance.

## How to Use

1. Download the `.json` workflow file you want to use
2. Open your n8n instance
3. Go to **Workflows** > **Import from File**
4. Select the downloaded JSON file
5. Configure any credentials or settings specific to your environment

## Workflow Categories

- **AI & Automation** - Workflows integrating AI services (OpenAI, Anthropic, etc.)
- **Data Processing** - ETL and data transformation workflows
- **Notifications** - Alert and notification automations
- **Integrations** - Third-party service integrations
- **Utilities** - Helper workflows and common patterns
