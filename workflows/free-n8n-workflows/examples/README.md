# Example Workflows

This folder contains example workflows to help you get started and understand the format.

## Hello World Webhook

**File:** `hello-world-webhook.json`

A simple starter workflow that demonstrates:
- Webhook trigger setup
- Setting response data
- Returning JSON responses

### How to Use

1. Import `hello-world-webhook.json` into n8n
2. Activate the workflow
3. Access the webhook URL (shown in the Webhook node)
4. You'll receive a JSON response with a greeting and timestamp

### No Credentials Required

This is a standalone example - no API keys or external services needed!
